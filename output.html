markdown-to-html-preview

TypeScript + ESM Node.js project that converts Markdown into HTML from the command line and provides an interactive browser preview mode.

Requirements

  • Node.js 20 or newer
  • npm

Project Structure

markdown-to-html-preview/
├─ src/
│  ├─ default-markdown.ts
│  ├─ file-service.ts
│  ├─ html-template.ts
│  ├─ index.ts
│  ├─ markdown-to-html.ts
│  └─ preview-server.ts
├─ .gitignore
├─ package.json
├─ README.md
└─ tsconfig.json

Install

npm install

Build the production output:

npm run build

Convert Markdown To HTML

Print generated HTML to the console:

npm run build
npm run convert -- README.md

Save generated HTML to a file:

npm run build
npm run convert -- README.md output.html

Preview In The Browser

Run the preview server with the built-in example:

npm run build
npm run preview

Optionally seed the editor from an existing Markdown file:

npm run build
npm run preview -- README.md

Then open:

http://localhost:3000

The preview page opens with a built-in Markdown example by default, shows a Markdown editor on the left, renders HTML on the right as you type, and lets you download both the current .md content and the generated .html file.

npm Scripts

npm run build
npm run convert -- README.md
npm run convert -- README.md output.html
npm run preview

Input And Output Example

Input Markdown:

# Hello

This is **Markdown**.

Output HTML:

<!doctype html>
<html lang="en">
  ...
</html>

Error Handling

The CLI exits with a non-zero status and prints an error message when:

  • the input path is missing
  • the input file does not exist
  • the input path is not a file
  • the Markdown file is empty

Coding Conventions

  • file and folder names use kebab-case
  • JavaScript variables and function names use camelCase
  • the project uses TypeScript and ESM modules

Extension Ideas
